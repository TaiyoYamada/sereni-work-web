# Web アプリ要件

管理者・支援員・閲覧者が利用する職員用 Web アプリ。ロール別権限の詳細は API リポジトリの docs/permissions.md を正とする。

## 主要機能

### 認証・アカウント管理
ログイン / ログアウト / メールアドレス認証 / パスワード再設定 / 職員招待 / アカウント停止 / 権限設定 / セッション管理

### 利用者管理
一覧・詳細 / 基本情報 / 希望職種 / 得意・不得意 / スキル / 必要な配慮 / 通勤条件 / 送迎要否 / 使用言語 / 訓練履歴 / 実習履歴 / 日報履歴 / 支援員評価確認

### 実習先企業管理
一覧・詳細 / 業種 / 実習内容 / 必要スキル / 対応可能な配慮事項 / 受け入れ可能人数・日程 / 勤務時間 / 担当者 / 所在地 / 持ち物・服装 / 緊急連絡先 / 過去の受け入れ実績

### 実習割当管理
割当作成 / 開始日・終了日設定 / 割当状況確認 / カレンダー表示（利用者別・企業別、日別・週別・月別） / 手動修正 / 確定 / 取消 / 中止 / 完了処理

### 割当自動提案
対象利用者・企業・期間の選択 / 条件重み設定（業務用語で） / ソルバー選択 / 実行 / 複数候補表示 / 提案理由表示 / 制約違反確認 / スコア内訳表示 / 候補比較 / 実行履歴 / 手動修正 / 最終確定

### 日報管理
一覧（未提出・提出済み・要確認） / 詳細 / 支援員コメント / 対応要否設定 / 面談必要フラグ / 検索 / 出力 / 修正履歴確認

### 支援員評価
実習態度 / 作業適性 / コミュニケーション / 配慮の適合度 / 継続勤務可能性 / 次回実習への所見 / 利用者自己評価との比較

### 教材管理
一覧 / 登録・編集 / 公開・非公開 / PDF・動画・画像・テキスト / 対応言語 / 閲覧状況確認

### ダッシュボード
実習中の利用者数 / 実習予定件数 / 未提出・要対応日報件数 / 実習完了率 / 利用者別進捗 / 企業別受け入れ状況 / 満足度・疲労度推移 / 職種別評価 / 割当実行履歴
